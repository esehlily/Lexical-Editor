# Lexical Editor

A lightweight and responsive rich-text editor built using HTML, CSS, and JavaScript. The editor provides an intuitive writing experience with text formatting capabilities and a clean user interface designed for content creation and editing.

## Features

* Rich text editing
* Text formatting options
* User-friendly interface
* Responsive design
* Lightweight and fast performance

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Getting Started

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Open the project folder.

3. Run ``npm run dev`` in your terminal.

## Project Purpose

This project was developed to explore the fundamentals of building a web-based text editor, focusing on user experience, text manipulation, and responsive interface design.
